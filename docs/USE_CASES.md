# 대표 유즈케이스

이 문서는 시스템의 대표적인 사용 시나리오를 설명합니다.

## 유즈케이스 1: 신규 배송 요청 최적화

1. 관리자가 `/admin` 페이지에서 차량/센터 정보를 등록.
2. 운영자가 `/plan` 페이지에서 신규 배송 작업(Job) 목록을 입력.
3. “최적화 실행” 버튼 클릭.
4. 백엔드에서 LLM/최적화 엔진을 이용해 경로 계산.
5. `/routes` 페이지에서 각 차량별 경로 및 KPI 확인.
6. `/dashboard`에서 주간 CO2, 거리, 이력 확인.

## 유즈케이스 2: KPI 모니터링 및 개선

1. 운영자가 `/dashboard` 페이지에서 최근 주간 CO2 추이 확인.
2. 비정상적으로 높은 주를 선택해 상세 데이터를 분석.
3. 필요 시 차량 구성/섹터 구성을 조정.
4. 여러 차례 최적화 실행 후 개선 효과를 대시보드에서 비교.
