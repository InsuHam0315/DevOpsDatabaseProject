# XAI (설명 가능한 AI) 설계 문서

이 문서는 `backend/services/xai.py` 중심의 설명 생성 로직을 설명합니다.

## 목적

- “왜 이 경로가 선택되었는지”를 사용자에게 이해하기 쉽게 설명합니다.
- 단순 숫자(KPI) 외에 의사결정 근거를 제공합니다.

## 입력

- 최적화 결과:
  - 차량별 경로, 작업 할당 정보.
- KPI:
  - 총 거리/CO2/시간, 평균 적재율 등.
- (옵션) LLM으로부터 받은 설명 텍스트.

## 출력

- 사용자용 설명 텍스트.
- 대시보드/계획 페이지에서 표시할 요약 정보.

예시(자연어):

> 전체 CO2 배출량을 최소화하기 위해 연비가 좋은 차량이 장거리 구간을 담당하도록 배치했습니다. 도심 구간은 교통 정체를 고려해 짧은 거리의 여러 작업을 묶어 처리했습니다.

## 구현 아이디어

- 규칙 기반 설명:
  - 예: “이 차량이 가장 많은 거리를 운행한 이유”, “특정 섹터가 특정 차량에 할당된 이유”.
- LLM 보조 설명:
  - 최적화 결과를 요약한 구조화 데이터 → LLM에 전달 → 자연어 설명 보완.
