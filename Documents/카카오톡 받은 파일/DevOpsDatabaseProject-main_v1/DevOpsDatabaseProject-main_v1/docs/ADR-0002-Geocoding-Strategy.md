# ADR-0002: 지오코딩 전략 (Kakao + Nominatim)

## 배경

주소 → 좌표(위경도) 변환이 필요하지만, 특정 주소는 Kakao Local API만으로 정확히 찾기 어려운 경우가 있습니다.

## 결정

- 기본적으로 Kakao Local API를 사용하되,
- 특정 케이스/실패 시 OpenStreetMap Nominatim 등 보조 지오코딩 서비스를 사용할 수 있도록 설계한다.

## 근거

- Kakao는 국내 주소에 강점이 있음.
- Nominatim은 글로벌 커버리지 및 오픈 데이터 기반 보완 수단 제공.

## 영향

- 지오코딩 모듈이 복수의 Provider를 다룰 수 있는 구조로 설계됨.
- 일부 주소는 매핑 테이블/캐시를 활용하여 성능을 개선할 수 있음.
